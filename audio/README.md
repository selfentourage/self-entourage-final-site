# Self Entourage Audio Assets

This folder is reserved for Self Entourage-owned audio files, including exported Suno tracks, founder voice notes, intro themes, store ambience, and private-program sound assets.

## Current audio standard

- Do not autoplay audio.
- Use user-controlled playback only.
- Prefer compressed `.mp3` files for website use.
- Keep homepage audio short, polished, and intentional.
- Only upload music Self Entourage owns or has the right to use commercially.

## Recommended file names

- `self-entourage-homepage-theme.mp3`
- `self-entourage-store-ambient.mp3`
- `self-entourage-founder-welcome.mp3`
- `black-crown-command-room.mp3`

## Future homepage player copy

Use a premium click-to-play callout such as:

> Listen to the Self Entourage Sound

The website should never surprise visitors with forced sound. The sound layer is meant to feel cinematic, sovereign, and optional.
